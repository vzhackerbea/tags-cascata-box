Display Block e Display Inline

Voltar ao início

Display Block

Display-block ocupa toda a tela em largura.

Propriedades:

  • Widht
  • Height
  • Padding
  • Margin
  • Border

Display Inline

Display-inline só ocupa o espaço necessário para existir.

Propriedades:

  • Padding
  • Margin
  • Border

Elementos

Display-block

  • H1
  • P
  • Div
  • Section
  • Main
  • Nav

Display-inline

  • A
  • Strong
  • Span
  • Img
